# 第二大脑浏览器插件安装说明

## 安装步骤

### 1. 打开 Chrome 扩展程序管理页面

在浏览器地址栏输入：
```
chrome://extensions/
```

### 2. 启用开发者模式

在页面右上角，找到「开发者模式」开关，
将其切换为 **开启** 状态。

### 3. 加载插件

1. 点击左上角的「加载已解压的扩展程序」按钮
2. 选择 `extension` 文件夹
3. 确认选择

### 4. 固定插件图标

安装成功后，插件图标会出现在工具栏右侧。
点击图标右边的 pin 图标，将「第二大脑」固定到工具栏。

## 使用方法

1. **打开任意网页**（知乎、微信公众号、小红书等）
2. **点击工具栏的「第二大脑」图标**
3. 弹出窗口会显示提取的标题、作者、平台
4. 点击「保存到第二大脑」
5. 等待自动提取和 AI 总结
6. 保存成功后，3秒自动刷新

## 设置选项

- **自动 AI 总结**：保存后自动调用 DeepSeek 总结内容
- 可以在弹窗中取消勾选

## 故障排除

### Q: 提取的内容不完整？

A: 尝试点击「重新提取」按钮，或刷新页面后重试。

### Q: 保存失败？

A: 请确保你已登录第二大脑账户（点击右上角头像登录），
扩展会默认发送到 `https://content-collector.vercel.app`。

如果需要使用本地开发版本，可以在扩展弹窗中设置自定义后端地址为 `http://localhost:3000`。

### Q: 如何卸载插件？

A: 在 `chrome://extensions/` 页面，找到「第二大脑」，
点击「移除」即可。

## 文件结构

```
extension/
├── manifest.json      # 插件配置
├── background.js      # 后台脚本
├── content.js         # 页面内容提取
├── popup.html         # 弹窗界面
├── popup.js           # 弹窗逻辑
└── icons/
    └── icon.svg       # 图标
```

## 注意事项

1. 插件需要配合第二大脑后端使用
2. 确保后端服务正在运行
3. DeepSeek API Key 已在 .env.local 配置
